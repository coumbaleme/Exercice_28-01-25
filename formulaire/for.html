formulaire de contact

Les Formulaires HTML permettent au user de'envoyer des données a un serveur

1. Structure d'un formulaire

Un formaulaire est definie par la balise form

Explicatin des attributs

name : cet attribut est utilesé pour edentifiér un champ du formulaire. Qand le formulaire est soumis , les données sont envoy"es sous form de pair clé-valeur ou le name est la clé

label for:l'attribut fordu label est utiliser pour associer un libellé a un champs via son id

2. type de champs

HTML offre plusieurs types de champs:

text--champs de text

email/--champs de email

pasword--champs de pasword

radio--champs de radio

texterea code>--zone de text

checkbox code>--case a cocher

numbercode>--entrée numerique

filecode>--upload de fichiers

submitcode>--button d'envoi

3. Exemples de Formulaires

Exemple avec différentes entrées :

<form>
    <label>Choisissez votre genre:</label>
    <input type="radio" name="genre" value="homme"> Homme
    <input type="radio" name="genre" value="femme"> Femme
    
    <label>Acceptez-vous les conditions ?</label>
    <input type="checkbox" name="conditions" required>
    
    <button type="submit">Valider</button>
</form>

4. SEO et les formulaires html

Les formulaires jouent un role important en seo,notament pour l'experience utilisateur et l'accessibilité

label et for: améliorent l'accessibilité et la lisibilté pour les moteurs de recherche

input=nam: les champs bien nommés aident les outiles d'analyse du site

l'attribut

Exemple de formulaire optimisé SEO :

<form action="traitement.php" method="POST">
    <label for="nom">Nom:</label>
    <input type="text" id="nom" name="nom" placeholder="Votre nom" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" placeholder="Votre email" required>
    
    <button type="submit">Envoyer</button>
</form>
