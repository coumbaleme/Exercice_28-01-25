Guide sur le RGPD

Le reglement général sur la protection des données (RGPD) est une réglementation européenne visant à protéger les données personnelles des citoyens

1. Principes fontamentaux du RGPD

licéité (fonforme ala loi), loyauté et transparenceles données doivent pour les finalités

limitation des finalités : Les données doivent eres collectees pour des finalités specifiqueet ligitime

limitation de conservationles données ne doiventpas etre conservees plus longtents que necessair.

intergrite etconfidentialitéles données doiventetre proteger contre tout acces non-autoriser.

2.Droit des utilisateurs

Les utiliateures disposes de plusiers droit sur les données

  • Droit a l'acces: coconsulter les donnéesdetenue par une organisation
  • Doit a la rectification madifier les données inexactes
  • Droit l'effacement démander la suppression des données
  • Droit a la probabilité:recuperer ses données sous format reutilisable
  • Droit a l'opposition s'opposer a l'utilissation des données
  • Droit a limitationrestreindre l'utilisation des données

3. obligation des entreprises

les organisations doiventrespecter plusieurs obligation

  • designer un DPO(deleguer a la protection des données ) si nzcessaire
