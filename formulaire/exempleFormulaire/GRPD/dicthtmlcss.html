Dictionnaire de Vocabulaire CSS et HTML

Ce guide couvre les termes clés et propriétés CSS et HTML les plus utilisés.

1. Bases du HTML

  • Balise : Un élément structurant une page HTML (ex: <p>, <div>).
  • Attribut : Une information supplémentaire sur une balise (ex: href, src).
  • Élément : Une balise et son contenu.
  • DOCTYPE : Définit le type de document.
  • Structure HTML : Contient les parties essentielles (<head>, <body>).

2. Balises Courantes

  • <a> : Lien hypertexte.
  • <img> : Image.
  • <table> : Tableau.
  • <form> : Formulaire.
  • <input> : Champ de saisie.
  • <div> : Conteneur générique pour la mise en page.
  • <section> : Regroupe un ensemble de contenu thématiquement lié.
  • <aside> : Contenu lié mais secondaire (ex: barre latérale).
  • <article> : Contenu indépendant (ex: un post de blog, une actualité).

3. Bases du CSS

  • Selector : Un élément HTML ciblé par un style.
  • Property : Une caractéristique CSS appliquée à un élément.
  • Value : La valeur d'une propriété CSS.
  • Class : Un groupe de styles appliqué à plusieurs éléments avec .nom-de-classe.
  • ID : Un identifiant unique appliqué avec #nom-id.

4. Propriétés CSS Courantes

  • color : Définit la couleur du texte.
  • background-color : Définit la couleur de fond d'un élément.
  • font-size : Définit la taille du texte.
  • margin : Définit l'espace extérieur autour d'un élément.
  • padding : Définit l'espace intérieur d'un élément.

5. Mise en Page

  • display : Contrôle l'affichage d'un élément (block, inline, flex).
  • position : Définit le positionnement d'un élément (static, relative, absolute).
  • flexbox : Une méthode moderne pour organiser les éléments (display: flex).
  • grid : Un système avancé de mise en page (display: grid).

6. Effets et Animations

  • transition : Crée des effets de transition fluides.
  • animation : Définit des animations CSS avancées.
  • transform : Permet des transformations (rotation, échelle, translation).

7. Responsive Design

  • media queries : Permettent d'adapter le style selon la taille de l'écran.
  • viewport : Définit la zone d'affichage sur mobile et desktop.
  • rem/em : Unités relatives pour le dimensionnement.
