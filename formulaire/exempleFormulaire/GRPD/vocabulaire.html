Cours sur la Mise en Page et l'Encapsulation en HTML

Introduction

La mise en page en HTML repose sur l'utilisation des balises de structure comme <div>, <section>, <aside>, et <article>. Ces balises permettent d'organiser les éléments d'une page de manière logique et flexible.

1. Qu'est-ce que l'Encapsulation en HTML ?

L'encapsulation consiste à regrouper plusieurs éléments HTML au sein d'un même conteneur. Cela permet :

  • De structurer le contenu de manière logique
  • De faciliter la gestion des styles CSS
  • D'améliorer la réutilisabilité des composants

2. Exemple de Mise en Page avec Encapsulation

Voici un exemple de mise en page utilisant des <div> pour organiser un site en trois parties : un en-tête, un contenu principal avec une barre latérale et un pied de page.

<div class="container">
    <div class="header">
        <h1>Titre du Site</h1>
    </div>
    
    <div class="main">
        <div class="sidebar">
            <h2>Menu</h2>
            <p>Liens et navigation</p>
        </div>
        
        <div class="content">
            <h2>Contenu Principal</h2>
            <p>Texte et images ici</p>
        </div>
    </div>
    
    <div class="footer">
        <p>Pied de page - Mentions légales</p>
    </div>
</div>

3. Organisation et Styles CSS

Pour donner un aspect visuel clair à cette structure, voici un exemple de styles CSS :

.container {
    width: 80%;
    margin: auto;
}
.header, .footer {
    background-color: #ddd;
    padding: 15px;
    text-align: center;
}
.main {
    display: flex;
    gap: 10px;
}
.sidebar {
    width: 30%;
    background-color: #f4f4f4;
    padding: 15px;
}
.content {
    width: 70%;
    background-color: white;
    padding: 15px;
}

Conclusion

Grâce à l'encapsulation et à une bonne organisation de la mise en page, il est possible de structurer un site de manière efficace et réutilisable. L'utilisation de <div>, <section>, et autres balises de structuration permet d'obtenir un rendu clair et adaptable sur différents supports.
